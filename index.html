Política de Privacidade

A aplicação GOODStudy QuickWords respeita a privacidade dos seus utilizadores e está comprometida com a proteção dos dados pessoais.

1. Recolha de dados

A aplicação não recolhe, armazena nem partilha quaisquer dados pessoais dos utilizadores.

2. Dados de crianças

A GOODStudy QuickWords é uma aplicação educativa adequada a crianças. Não são recolhidos dados pessoais de crianças ou adultos.

3. Permissões

A aplicação não solicita permissões sensíveis nem acede a funcionalidades do dispositivo que envolvam dados pessoais.

4. Terceiros

A aplicação não utiliza serviços de terceiros que recolham dados pessoais, tais como serviços de publicidade, analytics ou redes sociais.

5. Segurança

Como não são recolhidos dados, não existe armazenamento ou transmissão de informação pessoal.

6. Alterações a esta política

Esta política poderá ser atualizada no futuro. Quaisquer alterações serão publicadas nesta página.

7. Contacto

Para qualquer questão relacionada com esta política de privacidade, pode contactar-nos através do GitHub do projeto.
